# nmd_lungcells_2026

Analysis code accompanying the manuscript *"Long Read RNA Sequencing in Primary
Lung Cell Types Reveals Principles of Nonsense-Mediated Decay."*

The study characterizes nonsense-mediated decay (NMD) across four primary human
lung cell types (AT2, LAE, FB, MV) using **paired short-read and long-read
(PacBio Iso-Seq) RNA-sequencing** under SMG1 inhibition, plus a sequence-based
predictive model of NMD susceptibility.

This repository contains analysis code only. Data are available via the study
portal: https://nmd-lungcells.castaldilab.org/
