# data

Third-party reference datasets bundled for reproducibility. Please cite the
original sources.

- **`gerstberger_2014_rbp_census.csv`** — census of 1,542 human RNA-binding
  proteins. Gerstberger S, Hafner M, Tuschl T. *A census of human RNA-binding
  proteins.* Nat Rev Genet. 2014;15(12):829–845. Supplementary Table S3.
- **`encode_rbp_roster_vannostrand2020.csv`** — ENCODE eCLIP RBP roster.
  Van Nostrand EL, et al. *A large-scale binding and functional map of human
  RNA-binding proteins.* Nature. 2020;583:711–719 (ENCODE Consortium; released
  under the ENCODE data-use policy).

Tan et al. (2025) supplementary tables used by the transcript-level reanalysis
are under `../tan_reanalysis/data/` (tracked via Git LFS); see
`../tan_reanalysis/README.md`.
